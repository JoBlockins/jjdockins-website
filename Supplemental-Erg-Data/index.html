About This Project

Supplemental Erg Data captures supplemental real-time data from the Concept2 PM5 during a workout and analyzes it live so I can keep my stroke length and wattage exactly where they should be. After the session, it summarizes that extra data and merges it into the final workout summary the PM5 saves.

This project combines my passion for rowing with technology, giving me deeper insights into my training performance beyond what the standard PM5 display shows.

Real-Time Capture

Pulls supplemental data directly from the Concept2 PM5 during live workouts.

Live Analysis

Monitors stroke length and wattage in real-time to keep training on target.

Post-Session Summary

Merges supplemental data into the final workout summary for comprehensive review.
