About This Project

Bitcoin AI Agent Analysis is an automated system that pulls real-time and historical Bitcoin market data - including price, volume, volatility, trend strength, and key support/resistance levels - and then uses that information to evaluate what the market is most likely to do next.

This project explores the intersection of artificial intelligence and financial markets, using AI agents to process and interpret complex market signals in real time.

Real-Time Data

Pulls live Bitcoin price, volume, volatility, and trend data from market APIs.

AI-Powered Analysis

Uses AI agents to evaluate market conditions and predict likely market direction.

Key Levels Detection

Identifies support and resistance levels to provide context for market movement.

Analysis Dashboard

A live web dashboard that pulls real-time Bitcoin data from the CoinGecko API and runs technical analysis — including RSI, MACD, Bollinger Bands, support/resistance levels, and market regime classification — to evaluate current market conditions.

AI Trading Agent

An autonomous agent that scrapes the analysis dashboard, feeds the structured market data to Claude, and receives a BUY, SELL, or HOLD decision. It runs on a schedule, tracks a simulated portfolio, and logs every trade.
